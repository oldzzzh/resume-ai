 # 账号注册指南
 
 按优先级排列，先注册前面的。每一步我都给出了具体操作。
 
 ---
 
 ## 🔴 优先级高（今晚/明天必须注册）
 
 ### 1. GitHub（代码托管）
 **地址**：https://github.com
 **用途**：存放你的代码、管理版本、跟我协作
 **步骤**：
 1. 打开 github.com，点 Sign up
 2. 用户名建议用英文（如 yourname-dev）
 3. 邮箱用常用的，验证邮箱
 4. 选 Free（免费方案）
 5. 注册完后告诉我用户名，我把你加为这个项目的协作者
 
 **注意**：GitHub 不需要翻墙，国内可以直接访问
 
 ### 2. Vercel（前端托管部署）
 **地址**：https://vercel.com
 **用途**：把网站部署上线，给你的预览链接
 **步骤**：
 1. 用 GitHub 账号登录（点 Continue with GitHub）
 2. 授权后自动创建账号
 3. 选 Hobby（免费方案）
 4. 之后我部署时只需要你点一下确认
 
 **费用**：免费版每月100小时构建时间，个人项目完全够用
 
 ### 3. Lemon Squeezy（收款 —— 国内开发者首选）
 **地址**：https://www.lemonsqueezy.com
 **用途**：接受全球客户的美元付款，自动处理税务
 **为什么选它而不是 Stripe**：
 - Lemon Squeezy 是 Merchant of Record（商家记录），替你处理全球税务合规
 - 中国个人身份可以直接注册（Stripe 需要海外公司）
 - 支持 Payoneer 提现到国内银行卡
 - 5% + $0.50 手续费，无月费
 **步骤**：
 1. 打开 lemonsqueezy.com → Get Started Free
 2. 用邮箱注册
 3. 验证邮箱
 4. 进入 Settings → Store，设置商店信息
 5. 在 Settings → Payouts 中设置提现方式（推荐 Payoneer）
 6. 之后我做好产品后，你在这里创建产品定价，我集成到代码里
 
 **注意**：注册时用真实信息，后期提现需要身份验证
 
 ### 4. 域名（产品官网）
 **推荐**：Cloudflare 域名注册（成本价，不加价）
 **地址**：https://dash.cloudflare.com
 **步骤**：
 1. 注册 Cloudflare 账号
 2. 左侧菜单 → Registrar → 搜索域名
 3. 域名建议：简短、好记、.com 或 .io
    - 例如：resume-ai.com / covercraft.io / jobwrite.app
    - 年费大约 50-150 元
 4. 买完后告诉我，我帮你绑定到 Vercel

 ## 🟡 优先级中（第一周内完成）
 
 ### 5. 社交媒体账号
 
 **小红书**：注册账号，名字建议和你的项目主题相关
 - 如："独立开发者XX"、"零基础做软件"、"AI工具小白"
 
 **B站**：注册账号，同样名字
 - 发中视频（3-5分钟），内容更深度
 
 **抖音**：相比前两个优先级低一些
 - 发15-60秒短视频
 
 **X (Twitter)**：https://x.com
 - 做英文内容，"Building in public" 社群
 - 关注 @levelsio @Marc_Louvion @PieterLevels 等独立开发者
 
 ### 6. OpenAI API
 **地址**：https://platform.openai.com
 **用途**：AI简历优化等产品需要调用 GPT API
 **步骤**：
 1. 注册账号（需要海外手机号验证）
 2. 可以用 SMS 接码平台（如 sms-activate.org）
 3. 创建 API Key（Settings → API keys → Create new secret key）
 4. 首次充值 5美元（约36元）
 5. 把 API Key 保存好，之后我配置到项目里
 
 ### 7. Supabase（数据库）
 **地址**：https://supabase.com
 **用途**：存储用户数据、登录信息
 **步骤**：
 1. 用 GitHub 登录
 2. 创建一个新项目（免费版包含2个数据库）
 3. 之后我告诉你具体怎么配置
 
 ### 8. Payoneer（收款到国内银行卡）
 **地址**：https://www.payoneer.com
 **用途**：从 Lemon Squeezy 提现到国内银行卡
 **步骤**：
 1. 注册个人账号
 2. 填写真实身份信息
 3. 绑定国内银行卡
 4. 验证通过后，在 Lemon Squeezy 的 Payout 设置中选择 Payoneer
 
 **替代方案**：Wise (wise.com) 也可以用
